workTunkに関する情報を取得するスキルです。

下記ページにworkTunkが記載されているので、必要に応じて参照してください。
https://worktrunk.dev/worktrunk/

## 使い方

workTunkに関する情報を取得したいときに、以下のように入力してください。

```
mdget --extra-links https://worktrunk.dev/worktrunk/
```

上記コマンドを実行すると該当ページのメインコンテンツをmarkdownで取得できます
また文章の再度に該当のページに含まれているリンク集がまとまっているので、必要に応じでリンク先の情報も取得してください
