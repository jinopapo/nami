agent clinet protocolに関する情報を取得するスキルです。

下記ページにagent clinet protocolのschemaが記載されているので、必要に応じて参照してください。
https://agentclientprotocol.com/protocol/schema

## 使い方

agent clinet protocolに関する情報を取得したいときに、以下のように入力してください。

```
mdget https://agentclientprotocol.com/protocol/schema
```
