agent clinet protocolに関する情報を取得するスキルです。

## 使い方

agent clinet protocolに関する情報を取得したいときに、以下コマンドを実行してください

```
mdget --extra-links https://agentclientprotocol.com/protocol/schema
```

上記コマンドを実行すると該当ページのメインコンテンツをmarkdownで取得できます
また文章の再度に該当のページに含まれているリンク集がまとまっているので、必要に応じでリンク先の情報も取得してください
