# アプリ概要
- clineをチャットベースで操作するためのアプリケーション
- electron側でclineの操作や永続化などを行い、react側でユーザーインターフェースを提供する構成

# 技術ドキュメント
## [アーキテクチャ規約](docs/architecture-dependency.mmd)
- 実装や設計をする上で遵守すべきルールをまとめたドキュメント
- 実装や設計の際のガイドラインとして参照する

## [技術スタック](package.json)
- プロジェクトで使用している主要な技術やライブラリをまとめたドキュメント
- 実装や設計の際に、どんな技術が利用可能かを確認するときに参照する
- 作ろうとしているものが技術スタックに含まれているかを確認するときにも参照する

## [ディレクトリ構成](docs/architecture.md)
- プロジェクトのディレクトリ構成をまとめたドキュメント
- 各ディレクトリの役割や、どんなコードを格納するべきかを説明している
- 新しいコードをどこに追加すればいいかを確認するときに参照する
- 既存のコードがどこにあるかを確認するときにも参照する

# 設計思想
- ドメイン駆動設計を採用し、ドメインモデルを中心にアーキテクチャを構築する
- レイヤー単位で適切に抽象化されており、必要に応じて具体の処理をが終えるように単一責務の原則を重視
