# タスクワークスペース

## タスクワークスペースとは

各タスクで実装を行う環境

自動チェックを含めてタスクで行われるすべてのことはタスクワークスペースで行われる

## フロー

### 初期化

タスクが作成されたタイミングでプロジェクトワークスペースと同等の情報を持ったタスクワークスペースを作成する

このタイミングでタスクワークスペース専用のgit branchが切られる
git branchはタスク毎にユニークである必要がある

### 完了

タスクが完了したらプロジェクトワークスペースで使われているgit branchにタスクワークスペースのgit branchをマージする必要がある

もしコンフリクトが発生していたらユーザーにコンフリクトの解消を依頼する
